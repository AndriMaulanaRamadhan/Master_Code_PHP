Sejarah PHP.

PHP dibuat oleh Rasmus Lerdorf pada tahun 1994
pada awalnya Rasmus Lerdorf membuat sebuah program dalam bahasa C untuk mengelola halaman web pribadinya
itulah mengapa php adalah personal home page
halaman web yang dibuat menggunakan web form untuk terhubung kedalam database

PHP/FI, 1994 (Nama pertama PHP)
bisa juga diartikan personal home page/form interpreter

karena banyak orang yang minat maka php dibuat dalam bentuk open source
yaitu pada PHP Tools v.1, 1995

lalu selanjutnya PHP Tools v.2, 1997

pada tahun 1998 ada 2 orang yang membantu dalam project php yaitu :
zeev suraski dan andi Gutmans pada PHP 3, 1998

zeev dan andi membuat perusahaan bernama zend technologies
lalu nama PHP pun berganti menjadi PHP: Hypertext Preprocessor

PHP 4, 2000
sudah dibuatkan zend engine

PHP 5, 2004
sudah dibuatkan zend engine 2
sudah bisa object oriented programming (OOP)
dikenalkan juga PHP Data Object (PDO)

pada tahun 2008 perkembangan PHP4 diberhentikan

PHP 6 adalah project yang ditunda dan diabaikan

PHP 7, 2014
sudah dibuatkan zend engine 3

mascot PHP yaitu elePHPant diciptakan oleh vincent pontier

Karakteristik PHP :
1. Ekstensi filenya harus .php
2. ditulis dalam tag php (<?php) bisa juga disebut delimiter dan ditutup (?>)
3. bisa digunakan bersamaan dengan HTML
4. mengikuti bahasa C
