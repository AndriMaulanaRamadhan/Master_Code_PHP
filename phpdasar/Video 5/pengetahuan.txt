control flow (struktur kendali)

control flow adalah bagaimana alur di program kita ketika dibaca oleh interpreter php

control flow ada 2 pembahasan : pengulangan dan pengkondisian

pengulangan adalah ketika kita ingin mengerjakan sebuah blok program tanpa harus menulis ulang scriptnya

macam-macam pengulangan : for, while, do while, foreach

macam-macam pengkondisian : if else, if else if else, ternary, switch

perbedaan do while dan while : jika do while berkondisikan false maka jalankan 1 kali, kalau while jika kondisi false maka langsung terhenti
