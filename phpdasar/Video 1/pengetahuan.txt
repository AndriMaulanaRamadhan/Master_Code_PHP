PHP adalah bahasa pemrograman

10 Alasan mengapa kita harus belajar PHP :

1. Relatif mudah untuk pemula
2. Pre requisite (HTML + CSS) belajar PHP cukup hanya belajar HTML dan CSS sebagai dasar
3. Gratis dan open source
4. Dukungan komunitas yang besar
5. Dokumentasi yang lengkap
6. PHP dibuat khusus untuk web
7. Kebanyakan website yang ada saat ini dibuat dengan PHP
8. Banyak website besar dibuat dengan PHP
9. Content management System yang menggunakan PHP (Wordpress, Joomla!, Drupal, Moodle)
10. Framework PHP (Code igniter, Cake PHP, Yii Framework, Laravel, Symfony)

Yang harus disiapkan :

1. Code editor
2. Web server
3. Web Browser
4. Database Server
